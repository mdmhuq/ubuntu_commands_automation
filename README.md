# ubuntu_commands_automation

A Python CLI tool for managing Ubuntu packages using Typer.

## Features

- **Search for packages** by name
- **Fix broken dependencies**
- **Clean up packages** and remove unnecessary files
- **Find package dependencies and reverse dependencies**

## Requirements

- Python 3.7+
- [Typer](https://typer.tiangolo.com/)
- Ubuntu/Debian-based system
- Sudo privileges for package management commands
